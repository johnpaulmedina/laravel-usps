Laravel USPS v13.0

USPS Addresses API v3 (OAuth2) for Laravel 10–13

The legacy USPS Web Tools XML API was retired January 2026. This package uses the new OAuth2-based REST/JSON API at apis.usps.com.

Address Validation
Standardize and validate US addresses with ZIP+4
City/State Lookup
Resolve city and state from a ZIP Code
ZIP Code Lookup
Find ZIP Code for a given address
OAuth2 Built-in
Automatic token management with caching

Installation

composer require johnpaulmedina/laravel-usps

Configuration

php artisan vendor:publish --tag=usps-config

Add to your .env:

USPS_CLIENT_ID=your-client-id
USPS_CLIENT_SECRET=your-client-secret

Register at developer.usps.com for credentials.

Usage

Validate an Address

use Usps;

$result = Usps::validate([
    'Address' => '1600 Pennsylvania Ave NW',
    'City'    => 'Washington',
    'State'   => 'DC',
    'Zip'     => '20500',
]);

City/State Lookup

$result = Usps::cityStateLookup('33101');
// ['city' => 'MIAMI', 'state' => 'FL', ...]

ZIP Code Lookup

$result = Usps::zipCodeLookup([
    'streetAddress' => '1600 Pennsylvania Ave NW',
    'city'          => 'Washington',
    'state'         => 'DC',
]);
